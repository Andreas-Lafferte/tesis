Ellos contra nosotros: las percepciones del conflicto social

2.2 Métodos

Esta investigación utiliza un análisis transversal y en series de tiempo o tendencia para 45 países entre los años 1999, 2009 y 2019. Para especificar el efecto de variables contextuales sobre variables individuales (relaciones macro/micro), se estiman modelos multinivel. La principal razón es que estos modelos permiten analizar estructuras de datos complejas, en donde las unidades de análisis están anidadas en grupos ordenados jerárquicamente y en donde el valor esperado de una unidad dada en la variable dependiente depende del grupo(s) al que pertenece (Hox et al., 2017). De tal manera, estos modelos permiten plantear hipótesis sobre la relación entre variables contextuales, en este caso económicas e institucionales, con variables individuales como el índice de conflictos sociales percibidos \(PSCi\). Además, posibilitan plantear relaciones de interacción entre niveles lo cual es sumamente útil para analizar, por ejemplo, cómo cambia la relación entre la posición de clase y \(PSCi\) a partir de la influencia de la desigualdad económica (Hox et al., 2017).

En vista de que la base de datos posee una estructura de encuestas longitudinales comparativas (Schmidt-Catran & Fairbrother, 2016), los individuos encuestados en las diferentes olas son anidados tanto en países como en años (1999/2009/2019). Esto implica que el diseño multinivel de esta investigación distingue tres niveles: individuos = nivel 1; año-país = nivel 2; y país = nivel 3. En detalle, en estos modelos cada encuestado/individuo (nivel 1) es observado en un país específico y en un momento específico (nivel 2), y cada año-país es una única observación de un país (nivel 3) que se observa repetidas veces (Fairbrother, 2014). Esta estructura de tres niveles permite distinguir entre los efectos transversales de los países (nivel 3) y los efectos longitudinales dentro de los países (nivel 2). Además, incorpora efectos aleatorios en dichos niveles, lo cual es especialmente relevante cuando se estudian países ya que disminuye la heterogeneidad no observada entre estos grupos (Schmidt-Catran et al., 2019). En estos modelos se pueden incorporar variables contextuales en cualquiera de sus niveles, las cuales pueden ser características invariables de un país en el tiempo (“time-invariant”) como los marcos institucionales, o bien, pueden ser características de un país que varien en el tiempo (“time-variant”) como la tasa de desempleo o el nivel de desigualdad económica (Fairbrother, 2014). En el primer caso se busca conocer cómo una variable invariante en el tiempo \(x\) se asocia con los cambios en el tiempo de otra variable \(y\), mientras que en el segundo caso se busca conocer cómo los cambios en el tiempo de una variable \(x\) se asocian con los cambios en el tiempo de otra variable \(y\) (Schmidt-Catran & Fairbrother, 2016).

Para lograr lo anterior, se utilizan dos técnicas o extensiones de los modelos multinivel propuestas por Fairbrother (2014, pp. 124–127). Primero, se realiza un centrado al promedio del grupo/país de la variable contextual de interés, con el fin de diferenciar el componente transversal (media global de la variable de un país: \(\overline{Z}\)) del componente longitudinal (la diferencia entre la media global de la variable de un país y el valor de esa variable en un tiempo específico de un país: \(Z\) - \(\overline{Z}\)). Se incluye también una variable tiempo como control para revelar si la variable dependiente tiende a crecer más rápido en los países que experimentan un aumento más acelerado de la variable contextual de interés (Fairbrother & Martin, 2013). Segundo, se aplica la técnica de las curvas de crecimiento (“growth-curves”) a nivel país para conocer si el efecto de un valor determinado de alguna variable contextual que no varíe en el tiempo conduce a un cambio más rápido o más lento en la variable dependiente mediante un término de interacción entre la variable contextual y la variable tiempo (Fairbrother, 2014). Cabe mencionar que el interés principal de esta investigación recae en los efectos transversales de las características contextuales de los países, así como en los efectos de interacción entre niveles. Es probable que los cambios no abruptos en el tiempo de variables contextuales no impliquen cambios sustanciales en el \(PSCi\), considerando además que el lapso temporal de 20 años abarcado es relativamente corto como para captar efectos longitudinales que sean significativos (Schmidt-Catran et al., 2019).

En consecuencia, el modelo multinivel estimado es el siguiente (2.1):

\[\begin{equation} PSCi_{ijk} = \beta_{0} + \beta_{1}X_{ijk} + \beta_{2}\overline{Z}_{k} + \beta_{3}Z_{jkM} + \beta_{4}time_{jk} + \beta_{5}X_{ijk}\overline{Z}_{k} + \beta_{6}Z_{k}time_{jk} + v_{0k} + u_{0jk} + e_{ijk} \tag{2.1} \end{equation}\]

Donde:

  • \(PSCi_{ijk}\) = representa la estimación del índice de conflictos sociales percibidos
  • \(\beta_{0}\) = representa el gran intercepto del modelo
  • \(\beta_{1}\) = representa el efecto fijo de cada variable independiente de nivel individual
  • \(\beta_{2}\) = representa el efecto transversal de cada variable independiente de nivel contextual para cada país
  • \(\beta_{3}\) = representa el efecto longitudinal de cada variable independiente de nivel contextual para cada año-país
  • \(\beta_{4}\) = representa el efecto de la variable tiempo para controlar tendencias temporales simultáneas
  • \(\beta_{5}\) = representa el efecto de interacción entre niveles
  • \(\beta_{6}\) = representa el efecto de la curva de crecimiento (“growth-curve”)
  • \(v_{0k}\) = representa el efecto aleatorio de los países
  • \(u_{0jk}\) = representa el efecto aleatorio de cada año-país
  • \(e_{ijk}\) = representa el error estándar
  • \(X\) = representa cada variable de nivel individual
  • \(Z\) = representa cada variable de nivel contextual
  • \(\overline{Z}\) = representa el componente transveral a partir de la media global de la variable \(Z\) de nivel contextual para cada país
  • \(Z_{jkM}\) = representa el componente longitudinal a partir del centrado al promedio de la variable \(Z\) de nivel contextual para cada año-país
  • \(i\) = representa cada indiviuo
  • \(j\) = representa cada año
  • \(k\) = representa cada país
